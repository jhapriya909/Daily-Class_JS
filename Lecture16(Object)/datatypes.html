Datatype

There are two main types of datatype

1.) Primitive datatypes

2.) NON Primitive datatypes

We can clarify datatypes into 7 categories

    Primitive datatype

  1. number
  2. string
  3. boolean
  4. undefined
  5. null
  6. bigint
  7. symbol

Non primitive datatype

  1. Object

    Objects are further classifies into three categoriesa

  1. Object literals
  2. Arrays
  3. Fucntions
